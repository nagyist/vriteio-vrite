The GPT-3.5 extension integrates Open AI’s GPT-3.5 model in the Vrite editor as a block action.

## Installation

From the _Extensions_ side panel, in the _Available_ section, find _GPT-3.5_ and click **Install**. The GPT-3.5 extension has no configuration options.

![Vrite GPT-3.5 extension](https://assets.vrite.io/65017ed7b0e627e259623b8a/Wh-dMorMabthE-7QN8Z8j.png)

Upon successful installation, when your cursor is inside a **top-level paragraph**, you should see a new **Generate with GPT** block action appear in the editor.

![GPT-3.5 block action](https://assets.vrite.io/65017ed7b0e627e259623b8a/GldPAk8p46cgTaENCJNTz.png)

## Usage

Click the **Generate with GPT** block action to open a menu. Here you can type in your prompt and click _Generate_ for the GPT-3.5 to start generating the content.

![GPT-3.5 generating content](https://assets.vrite.io/65017ed7b0e627e259623b8a/HlW2G7hxGnWXDwNQajAJ0.png)

While the generation is ongoing, you can use the _Stop_ button to stop it.

To make GPT-3.5 take action on an existing piece of content (e.g. _”Make it shorter”_), check the _Include context_ option — it’ll include the active paragraph in the prompt.
